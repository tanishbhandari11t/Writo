## 🚀 Overview

**WRITO** is a professional-grade SaaS engine designed to bypass AI detection and inject human personality into machine-generated content. Built with a "Stark Tech" aesthetic, it features real-time telemetry, visual diffing, and a dual-AI orchestration layer (Gemini + Groq) to ensure high-fidelity humanization.

## ✨ Key Features

- **🧠 Humanization Formula**: Uses a proprietary `Simple words + Varied sentences + Emotion + Real-world examples + Contractions` logic.
- **🛡️ Detector Bypass**: Specifically engineered to eliminate "AI Fingerprints" and banned clichés.
- **🎨 Stark Aesthetic**: High-fidelity glassmorphism with an immersive isometric cubic background.
- **🔍 Visual Diffing**: Real-time tracking of improvements with a "Clean View" highlight (green = improved).
- **⚡ Dual-AI Core**: Primary processing via **Gemini 1.5 Flash** with an automatic failover to **Groq (Llama 3.3 70B)**.
- **📈 Real-time Telemetry**: Live tracking of characters processed, request counts, and daily active users.
- **📋 One-Click Portability**: Clean-text clipboard utility for instant deployment.

## ⚙️ Project Setup

### 1. Prerequisites
- Node.js (v18+)
- API keys for [Google Gemini](https://aistudio.google.com/) and [Groq](https://console.groq.com/).

### 2. Installation
```bash
git clone https://github.com/tanishbhandari11t/Writo.git
cd Writo
npm install
```

### 3. Environment Configuration
Create a `.env` file in the root directory:
```env
VITE_GEMINI_API_KEY=your_gemini_key
VITE_GROQ_API_KEY=your_groq_key
```

### 4. Run Development Server
```bash
npm run dev
```
